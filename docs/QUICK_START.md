## 🎨 UI功能说明

### 输入框功能按钮

从左到右：
1. **🧩 技能按钮**（蓝色）- 选择特定技能（图像生成、活动策划等）
2. **📋 指令按钮**（紫色）- 选择预设工作流模板
3. **📤 发送按钮**（主色） - 发送消息

### 选中状态显示

- **技能选中**：输入框内左侧显示技能名称（蓝色标签）
- **指令选中**：输入框内左侧显示指令名称（紫色标签）
- 悬停标签可点击 ✕ 按钮清除选择

## ⚙️ 高级配置

### 修改模型配置

编辑 `src/services/modelConfig.ts`：

```typescript
export const MODEL_CONFIG = {
  general_chat: {
    model: 'gpt-4o-mini',  // 修改为你想用的模型
    temperature: 0.7,
    max_tokens: 2000
  },
  // ... 其他配置
};
```

### 添加新的意图类型

1. 编辑 `src/services/intentClassifier.ts`
2. 在 `IntentType` 添加新类型
3. 在 `fallbackIntentDetection` 添加关键词规则
4. 在 `ChatPage.tsx` 的 `handleIntelligentSend` 添加处理逻辑

## 🐛 常见问题

### Q: 为什么意图识别不准确？

A: 可能原因：
1. 关键词不够明显 → 系统会调用LLM进行精确识别
2. 消息太短 → 尝试更详细的描述
3. 模型响应慢 → 检查网络和API配置

### Q: 如何强制使用某个工具？

A: 有两种方法：
1. **使用技能按钮**：点击 🧩 选择具体技能
2. **使用指令模板**：点击 📋 选择预设工作流

### Q: Todo功能什么时候触发？

A: 满足以下条件之一：
1. 选择了指令模板
2. 消息被识别为 `need_workflow` 意图
3. 消息长度超过50字且包含复杂任务关键词

### Q: 如何查看使用了哪个模型？

A: 打开浏览器控制台，查看网络请求中的 `model` 参数
